### 环境搭建调试和发布:
+ **初始环境**:

  * > 设置go modules    Proxy 建议使用: https://goproxy.cn

  + > go mod tidy

* **调试与发布**:

     * debug环节，在终端输入air即可调试

        > air  或 air -c  ./.air.conf

        

     * build环节，在终端输入make即可构建go二进制文件

        > make 或 make all

* **other**: 支持直接使用swagger生成API, 需要在api接口附近添加
  
  * swag init
